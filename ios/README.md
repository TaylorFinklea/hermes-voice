# iOS

SwiftUI app. Project generated via [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `HermesVoice/project.yml`.

```bash
cd HermesVoice
xcodegen generate    # regenerates HermesVoice.xcodeproj
open HermesVoice.xcodeproj
```

The `.xcodeproj` is intentionally not checked in вЂ” `project.yml` is the source of truth. Run `xcodegen generate` after pulling.

## Microphone permission

The `NSMicrophoneUsageDescription` in `project.yml` is what iOS shows on the first record attempt. If you change the wording, regenerate the project.

## Push-to-talk gesture

The button accepts both:
- **Hold** for в‰Ґ350ms в†’ release sends. Walkie-talkie.
- **Brief tap** (<350ms) в†’ latches recording on. Tap again to stop.

Threshold lives in `PushToTalkButton.tapThreshold`.
